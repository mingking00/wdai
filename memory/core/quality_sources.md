## 内容偏好特征

### 关键词优先级
```

P1 (高兴趣): 开发工具、架构、系统
P2 (有兴趣): 效率方法、认知科学、游戏引擎
P3 (一般): 其他技术内容
```

### 视频时长偏好
- 短视频 (<5min): 工具介绍、快速演示
- 中视频 (5-15min): 深度解析、教程
- 长视频 (>15min): 系统学习、综合内容

### 内容类型
1. **技术解析** - Agent架构、系统设计
2. **工具推荐** - AI编程工具、效率工具
3. **实践案例** - 真实项目、代码演示
4. **前沿追踪** - 论文、新研究、开源项目
