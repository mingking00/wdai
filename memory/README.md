## 🚀 使用方式

### 新增记忆
```bash
cp .templates/memory_template.md short_term/2026-03/my-event.md
```

### 定期维护（推荐添加到 crontab）
```bash
# 每周一凌晨 2 点：日常维护 + 压缩
0 2 * * 1 cd /root/.openclaw/workspace/memory && python3 .tools/memory_maintenance.py && python3 .tools/memory_compress.py

# 每月 1 号：质量评估 + 冲突检测
0 3 1 * * cd /root/.openclaw/workspace/memory && python3 .tools/memory_quality.py && python3 .tools/memory_conflict.py
```

### 手动检索
```bash
python3 .tools/memory_search.py
```
