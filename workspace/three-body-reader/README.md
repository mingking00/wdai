# 📖 三体交互阅读器

> Vibero风格的交互式PDF阅读器，专为《三体》小说优化

---

## ✨ 功能特性

### 📚 智能导航
- **章节导航**: 按部/章组织，点击快速跳转
- **进度追踪**: 当前章节高亮显示
- **页码跳转**: 支持输入页码直接跳转

### 👤 角色系统
- **角色面板**: 主要角色快速查看
- **详情弹窗**: 点击角色查看完整介绍
- **角色关系**: 显示角色的关键事件和性格特点

### 🕐 时间线
- **时代划分**: 文革时期 → 现代 → 危机纪元 → 广播纪元 → 宇宙末日
- **事件标注**: 每个时代的关键事件

### 💡 核心概念
- **概念标签**: 智子、黑暗森林、水滴、降维打击等
- **概念详解**: 点击查看详细解释和引用
- **名言展示**: 每个概念的经典语录

### 📖 阅读体验
- **PDF渲染**: 基于PDF.js的高质量渲染
- **缩放控制**: 支持放大/缩小
- **键盘快捷键**: 方向键、空格键翻页
- **拖拽上传**: 支持拖拽PDF文件

---

## 🚀 使用方法

### 1. 准备PDF文件
获取《三体》英文版PDF文件（或其他语言版本）

### 2. 打开阅读器
在浏览器中打开 `index.html`

```bash
# 方式1: 直接双击打开
open index.html

# 方式2: 使用本地服务器（推荐）
cd workspace/three-body-reader
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

### 3. 上传PDF
- 点击"上传PDF"按钮
- 或直接将PDF文件拖拽到页面

### 4. 开始阅读
- 使用左侧章节导航快速跳转
- 点击角色查看人物介绍
- 点击概念标签了解科幻设定
- 使用时间线理解故事脉络

---

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `←` / `PageUp` | 上一页 |
| `→` / `PageDown` / `空格` | 下一页 |
| `Home` | 第一页 |
| `End` | 最后一页 |
| `+` | 放大 |
| `-` | 缩小 |

## 🛠️ 技术栈

- **PDF渲染**: [PDF.js](https://mozilla.github.io/pdf.js/) (Mozilla)
- **样式**: 纯CSS3，深色主题
- **交互**: 原生JavaScript，无框架依赖

---

## 📝 注意事项

1. **跨域问题**: 由于浏览器安全限制，建议使用本地服务器打开
2. **PDF格式**: 支持标准PDF文件，扫描版PDF可能无法选中文本
3. **章节映射**: 当前使用预定义的章节页码，实际PDF可能有所不同
4. **字体显示**: 确保系统支持中文字体
