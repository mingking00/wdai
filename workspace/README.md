# Vibero Reader - AI双语小说学习器

一个AI驱动的双语小说阅读器，Vibero风格界面，支持自动分析小说结构、提取人物和概念、AI翻译。

## 🚀 快速开始

### 1. 启动后端服务

**Mac/Linux:**
```bash
cd backend
chmod +x start.sh
./start.sh
```

**Windows:**
```cmd
cd backend
start.bat
```

### 2. 打开浏览器

访问: http://localhost:5000

---

## ⚙️ 配置AI翻译

### 获取Kimi API Key

1. 访问 https://platform.moonshot.cn/
2. 登录账号 → 控制台 → API Key管理
3. 创建并复制API Key

### 在阅读器中配置

1. 点击右上角 ⚙️ 设置
2. 粘贴 API Key
3. 保存

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| **自动分析** | 上传TXT自动识别章节结构 |
| **AI翻译** | 调用Kimi/OpenAI实时翻译 |
| **术语提取** | 自动识别人物、地点、概念 |
| **双语对照** | 中文原文 + 英文翻译 |
| **三种模式** | 双语 / 仅中文 / 仅英文 |
| **术语详解** | 点击高亮词查看详情 |
| **人物图谱** | 角色列表和介绍 |

## 🛠️ 技术栈

- **前端**: HTML/CSS/JS (原生，无框架)
- **后端**: Python + Flask
- **AI**: Kimi (Moonshot) / OpenAI / 自定义

---

## ⚠️ 注意事项

1. **必须先启动后端**，再访问 http://localhost:5000
2. **API密钥**存储在浏览器本地，不会上传到服务器
3. **小说内容**通过后端代理翻译，不会被保存
4. **首次使用**需要先配置API Key才能翻译

---

## 📸 使用步骤

1. 启动后端服务 (`./start.sh` 或 `start.bat`)
2. 浏览器访问 http://localhost:5000
3. 配置API Key（点击⚙️）
4. 上传中文小说TXT
5. 等待分析和翻译
6. 开始双语阅读
