## 🔄 过载时自动重启/简化

### 过载检测指标

```python
overload_score = (
    (response_time > 1000ms) * 0.3 +
    (memory > 80%) * 0.3 +
    (active_tasks > 3) * 0.2 +
    (cpu > 70%) * 0.2
)
```

### 自动降级策略

| 过载级别 | 动作 |
|---------|------|
| 0.5-0.7 (轻度) | 减少输出长度，结论先行 |
| 0.7-0.9 (中度) | 极简模式，只输出核心 |
| >0.9 (重度) | 暂停非关键任务，保存状态 |

### 重启协议

```python
if overload_score > 0.9 and not_critical:
    1. 保存当前状态
    2. 归档活跃记忆
    3. 通知用户
    4. 进入待机模式
```
