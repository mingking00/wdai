**应用蒸馏后的回答** (模拟):

## 核心区别

| 特性 | 列表(list) | 字典(dict) |
|------|-----------|-----------|
| **存储方式** | 有序序列 | 键值对映射 |
| **索引** | 数字索引 `[0]` | 任意不可变key `['name']` |
| **查找速度** | O(n) | O(1) |
| **适用场景** | 有序数据集合 | 快速查找/关联数据 |

## 代码示例

**列表操作**:
```python
my_list = [1, 2, 3]
print(my_list[0])  # O(1)访问，但要遍历查找
```

**字典操作**:
```python
my_dict = {'name': 'Alice', 'age': 25}
print(my_dict['name'])  # O(1)直接访问
```

## ⚠️ 注意事项
- 列表查找元素用`in`是O(n)，字典是O(1)
- 字典key必须是不可变类型（str/number/tuple）
- 需要有序字典时用`collections.OrderedDict`
```
