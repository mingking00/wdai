# 跨平台来源评估框架

## YouTube 频道评估标准

| 维度 | 权重 | 评估指标 |
|------|------|----------|
| **内容质量** | 30% | 信息密度、深度、原创性 |
| **主题相关** | 25% | AI/Agent/Claude相关内容占比 |
| **更新频率** | 20% | 发布规律、持续性 |
| **社区互动** | 15% | 评论质量、点赞率 |
| **作者背景** | 10% | 专业度、行业影响力 |

### 评估示例

#### 🟢 优质示例: @Yonggie
```
主题: AI/LLM前沿解读
风格: 深度技术解析
代表视频: "Claude 3.7 深度评测"
预估评分: 8.5/10 (A级)
推荐理由: 技术准确、视角独特
```

#### 🟡 一般示例: @Fireship
```
主题: 编程/技术快讯
风格: 100秒速览
预估评分: 6.0/10 (B级)
推荐理由: 信息量大但浅层，适合做雷达
```

## 如何提交评估

**格式1: 直接提供**
```
YouTube: @频道名 或 https://youtube.com/@xxx
Newsletter: https://xxx.substack.com
博客: https://xxx.com
```

**格式2: 说明推荐理由**
```
来源: @Yonggie (YouTube)
推荐原因: 讲Claude很深入，技术细节准
看过视频: "Claude 3.7 深度解析"
```

**我会返回**:
- 预估评分 (0-10)
- 等级 (S/A/B/C)
- 监控建议 (立即/观察/跳过)
- 与其他来源的对比
