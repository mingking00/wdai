一、企业基本信息

（一）企业概况

联想集团成立于1984年，由中科院计算所投资20万元人民币、11名科技人员创办，总部位于中国北京。作为全球领先的科技公司，联想主要从事开发、制造并销售PC、服务器、智能手机、平板电脑等产品。2014年完成对摩托罗拉移动的收购后，已成为年收入569亿美元的全球化科技巨头，位列《财富》世界500强第248名，服务遍布全球180个市场。

（二）企业规模

• 生产基地：北京、上海、惠阳三大生产基地
• 供应商网络：全球300多家供应商
• 客户渠道：国内5000多家客户渠道
• 行业地位：2013年起成为全球最大PC生产厂商

（三）组织架构

联想采用事业部制组织架构，设有：
• PC业务集团
• 移动业务集团  
• 企业级业务集团
• 云服务业务集团

二、采购物品类型

（一）生产性物料（占采购成本60-70%）

1. 核心零部件
• 半导体芯片（CPU、GPU、内存芯片）
• 显示屏（LCD、OLED面板）
• 电池及电芯
• 主板及PCB板
特点：技术门槛高、价格波动大

2. 结构件
• 机壳、键盘、散热模组
• 连接器、线缆
特点：标准化程度高，但差异化设计空间大

3. 原材料
• 各类金属、塑料、化工原料
采购方式：长期协议（LTA）

（二）非生产性物料（占采购金额15-20%）

• 研发设备、测试仪器
• 软件授权、广告服务
• 办公用品、员工福利
• IT设备、云服务
特点：品类繁杂、供应商分散

三、采购流程

（一）采购流程图

[需求预测] → [供应商评估] → [询价比价] → [合同签订] → [订单执行] → [验收入库] → [付款结算]
      ↑                                                              ↓
      └────────────── 绩效评估反馈 ←───────────────────────────────────┘

（二）流程各环节说明

1. 需求预测与计划制定
• 采用"销售预测驱动"模式
• 通过SCI系统整合销售预测、客户订单、市场趋势
• 每天更新目标和计划
• 预测准确率：85%以上

2. 供应商选择与评估
• 准入评估：生产能力、质量体系（ISO9001）、环境责任（ISO14001）
• 风险评估：95%供应商每两年进行RBA审核
• 分类管理：战略/关键/一般三级差异化管理

3. 采购执行
• 采购申请 → 三级审批 → 询价比价 → 合同签订 → 订单下达 → 到货验收 → 付款结算
• 紧急订单：绿色通道24小时完成审批

4. 特殊采购模式
• VMI（供应商管理库存）：库存从14天降至5天，成本降低60%
• CTO（客户定制）：按订单生产+安全库存结合
• 电子招标：2023年节约12亿元

四、采购部门设置和人员配置

（一）组织架构

采用"集中决策、分散执行"模式：

1. 全球采购中心（GPO）
地点：美国北卡罗来纳州
职能：战略寻源、供应商管理、采购策略制定
团队规模：
• 生产采购团队：约150人
• 一般采购团队：约80人
• 供应商管理团队：约30人

2. 区域采购中心
地点：中国（北京、上海、深圳）、美国、巴西、匈牙利、印度
职能：本地采购执行、供应商协调、物流管理
中国区团队：约200人

3. 跨职能协同
机制：品类委员会（Commodity Council）
参与方：采购、研发、质量、财务
频率：每月例会

五、采购过程中的挑战和解决方案

（一）挑战一：价格波动风险

问题描述：
• 半导体、显示屏价格波动超200%
• 汇率波动（美元兑人民币年波动5-8%）
• 地缘政治影响（中美贸易摩擦芯片出口管制）
• 2023年芯片短缺导致紧急采购成本增加约5亿元

解决方案：
1. 双源供应策略
   • TOP50关键物料均有两个以上供应商
   • 降低单一供应商依赖风险

2. 战略库存管理
   • 建立6个月安全库存
   • 2022年芯片短缺期间保障95%订单交付（行业平均80%）

3. 本地化采购
   • 中国区80%生产采购支出来自本地供应商
   • 减少汇率波动和地缘风险
   • 降低碳排放

（二）挑战二：需求不确定性

问题描述：
• 客户定制（CTO）订单占比40%
• 标准化与差异化矛盾
• 技术迭代快（CPU/内存12-18个月换代）
• 旧型号快速贬值（30-50%降价）

解决方案：
1. 混合生产模式
   • 1-2天成品安全库存（畅销型号）
   • 按订单生产快速响应
   • 效果：缺货率从5%降至2%，库存周转天数从18天降至12天

2. VMI实时补货
   • 供应商根据实际销售数据自动补货
   • 替代传统"按预测补货"
   • 效果：库存周转天数从14天降至5天，缺货率<1%

3. 绿色供应链管理
   • 建立GSCDM数据平台
   • 95%供应商通过RBA审核
   • 获CDP"供应链脱碳先锋奖"

（三）挑战三：供应链复杂性

问题描述：
• 300多家供应商遍布30多个国家
• 海运、空运、陆运多种运输方式
• 二级、三级供应商管控能力弱

解决方案：
1. 供应商协同平台
   • 供应商门户系统在线协同
   • 功能：在线报价、订单查询、库存共享、质量数据交换、电子对账
   • 使用率90%，供应商满意度4.2/5.0

2. 早期供应商参与（ESI）
   • 每季度与战略供应商召开QBR会议
   • 效果：准时交付率从90%提升至95%，质量问题投诉减少40%

3. 数字化风险监控
   • 7×24小时供应链监控中心
   • 供应链风险预警AI模型（提前30天预测，准确率>80%）

六、总结与建议

（一）总结

联想采购体系的核心竞争力：
1. 一体化运作：打破部门墙，实现端到端协同
2. 分层管理：战略/关键/一般三级差异化配置资源
3. 数字化领先：SCI系统实现预测智能化、采购自动化、风险可视化
4. 成本控制：2023年采购成本节约率8.5%，库存周转率12次/年（行业平均8-10次）

（二）改进建议

1. 对联想：从成本中心到价值中心
   • 前置参与产品设计，通过VA/VE降低总成本
   • 设立供应商创新基金
   • 建立采购价值核算体系

2. 对行业启示
   • 数字化成功的关键是流程重构，而非简单上系统
   • 人机协同：AI负责预测，人负责关键决策

3. 对学生启示
   • 采购是战略职能，不是"买东西"
   • 未来核心竞争力：数据分析能力 + 跨部门协作 + 供应商战略管理
