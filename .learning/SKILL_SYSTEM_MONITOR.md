## 🔧 沉淀的技能详解

### 技能1: 系统信息快速收集

**核心能力**: 通过/proc和系统命令快速了解系统全貌

**关键技术**:
```bash
# 系统信息
/proc/cpuinfo      → CPU详细信息
/proc/meminfo      → 内存使用情况
/proc/loadavg      → 系统负载
/proc/net/dev      → 网络流量

# 系统命令
uname -r           → 内核版本
lsb_release        → 发行版信息
free -h            → 内存使用
df -h              → 磁盘使用
```

**应用场景**: 
- 新系统环境评估
- 问题初步排查
- 系统文档记录

**有效性**: ✅ 高 (已在多个系统测试)

---

### 技能2: 性能监控与趋势分析

**核心能力**: 实时监控系统资源，识别性能瓶颈

**关键技术**:
```bash
# CPU监控
top -bn1 | grep "Cpu(s)"  # 即时CPU使用率

# 内存监控
free | awk计算使用率       # 内存使用百分比

# 负载监控
uptime | awk提取load average

# IO监控
iostat -x 1 1             # 磁盘IO统计
```

**数据持久化**: CSV格式日志，便于后续分析

```csv
timestamp,cpu_usage,mem_usage,load_avg,disk_io
2026-03-10 06:54:00,15.2,28.5,0.18,0.5
```

**应用场景**:
- 性能问题排查
- 容量规划
- 趋势分析

**有效性**: ✅ 高

---

### 技能3: 异常自动检测

**核心能力**: 基于阈值自动发现系统异常

**检测维度**:
| 指标 | 阈值 | 检测方法 |
|------|------|----------|
| CPU | 80% | top命令 |
| 内存 | 85% | free命令 |
| 负载 | 核心数 | uptime |
| 磁盘 | 90% | df命令 |
| 僵尸进程 | >0 | ps aux |

**告警机制**:
- 实时输出到终端
- 持久化到日志文件
- 可扩展为邮件/钉钉通知

**应用场景**:
- 7x24小时监控
- 自动故障发现
- 预防性维护

**有效性**: ⚠️ 中 (需要调优阈值)

---

### 技能4: 进程分析与问题定位

**核心能力**: 快速定位资源消耗异常的进程

**分析维度**:
```bash
# CPU占用TOP
ps aux --sort=-%cpu

# 内存占用TOP
ps aux --sort=-%mem

# 运行时间
ps aux --sort=-etime

# 线程数
ps -eLf |统计线程数
```

**输出优化**:
- 格式化输出，易读
- 关键信息突出
- 支持快速扫描

**应用场景**:
- 系统卡顿排查
- 资源泄漏定位
- 恶意进程发现

**有效性**: ✅ 高

---

### 技能5: 网络诊断与排查

**核心能力**: 快速诊断网络连接问题

**诊断项目**:
```bash
# 接口状态
ip -s link

# 端口监听
ss -tlnp

# 连接统计
ss -s

# 连通性测试
ping 8.8.8.8
nslookup
```

**应用场景**:
- 网络不通排查
- 端口冲突检查
- 流量异常分析

**有效性**: ⚠️ 中 (需要root权限)
