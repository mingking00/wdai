## 💡 第一性原理总结

**计算机是什么？**
- 硬件: 晶体管 → 逻辑门 → ALU → CPU → 计算机
- 软件: 指令序列操作硬件

**为什么要从Nand开始？**
- Nand是硅芯片的"原子"
- 理解Nand=理解计算机的物理基础
- 从Nand到俄罗斯方块=理解完整计算机系统

**学习顺序的本质：**
- 不是"先学什么后学什么"的技巧
- 而是"理解基础才能理解上层"的物理规律
