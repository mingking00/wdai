一、企业简介

（一）企业基本信息

联想集团成立于1984年，由中科院计算所投资20万元人民币、11名科技人员创办。作为全球领先的科技公司，联想主要从事开发、制造并销售可靠的、安全易用的技术产品及优质专业的服务。2014年完成对摩托罗拉移动的收购后，联想集团已成为一家年收入569亿美元的全球化科技巨头，位列《财富》世界500强第248名，服务遍布全球180个市场数以百万计的客户。

（二）企业规模与行业地位

联想集团在全球拥有300多家供应商，管理国内5000多家客户渠道，内部设有北京、上海和惠阳三个主要生产基地。作为全球电脑市场的领导企业，联想自1996年起电脑销量一直位居中国国内市场首位，2004年收购IBM PC事业部，2013年成为全球最大的PC生产厂商。目前主要生产台式电脑、笔记本、服务器、智能手机、平板电脑等产品。

（三）组织架构

联想集团采用事业部制组织架构，设有PC业务集团、移动业务集团、企业级业务集团、云服务业务集团四个相对独立的业务集团。在供应链管理方面，联想将采购、生产、分销以及物流整合成一个统一的系统，实现了从战略层到执行层的统一协调。

三、采购物品和流程分析

（一）采购物品类型

联想集团的采购物料主要分为以下两大类：

1. 生产性采购物料

（1）核心零部件：包括半导体芯片（CPU、GPU、内存芯片等）、显示屏（LCD、OLED面板）、电池及电芯、主板及PCB板等。这类物料占总采购成本的60-70%，技术门槛高、价格波动大。

（2）结构件：包括机壳、键盘、散热模组、连接器、线缆等。这类物料标准化程度高，但差异化设计空间大。

（3）原材料：包括各类金属、塑料、化工原料等基础材料。主要通过长期协议（LTA）方式采购。

2. 非生产性采购物料

包括研发设备、测试仪器、软件授权、广告服务、办公用品、员工福利、IT设备、云服务、办公场地租赁等。这类物料采购金额占比约15-20%，但品类繁杂、供应商分散。

（二）采购流程

联想集团采用"一体化运作体系"，将采购流程与生产、分销、物流整合成统一系统。具体流程如下（见图1）：

1. 需求预测与计划制定

联想采用"销售预测驱动"的采购模式。通过供应链智能控制塔（SCI）系统，整合销售预测、实时客户订单及市场趋势分析，每天根据市场变化更新目标和计划。系统利用AI和大数据技术提高预测准确性，使决策时间缩短50%-60%，预测准确率提升至85%以上。

2. 供应商选择与评估

（1）供应商筛选：建立严格的供应商准入机制，对供应商的生产能力、质量管理体系（ISO9001）、环境责任（ISO14001）等进行全面评估。新供应商准入周期约3-6个月。

（2）风险评估：要求95%（按采购金额计）的供应商每两年进行一次RBA（责任商业联盟）VAP审核或同等独立第三方审核。对于高风险供应商，实施现场审计。

（3）分类管理：根据供应商的重要性和风险水平，将供应商分为战略供应商（TOP20，占采购额60%）、关键供应商（占采购额30%）、一般供应商（占采购额10%）三类，实施差异化管理。

3. 采购执行流程

（1）采购申请：各业务部门通过ERP系统提交采购申请，经部门经理、财务、采购三级审批后进入采购流程。

（2）询价比价：通过供应商门户系统发布询价单（RFQ），收集至少3家供应商报价。对于战略物料，采用电子招标（e-Auction）方式，平均可降低采购成本5-8%。

（3）合同签订：与选定供应商签订采购合同，明确交付条款（Incoterms 2020）、质量要求（AQL标准）、价格机制（季度调价）、违约责任等。

（4）订单下达：通过ERP系统下达采购订单（PO），供应商确认后进入执行阶段。紧急订单可通过"绿色通道"在24小时内完成审批。

（5）到货验收：IQC（来料质量控制）对采购物料进行检验，包括外观检查（100%）、功能测试（AQL 1.0）、可靠性测试（抽样）。不合格品启动MRB（物料评审委员会）流程。

（6）付款结算：根据合同约定的付款条件（通常是到货后60-90天）进行结算。战略供应商可享受提前付款折扣（2/10, n/60）。

4. 特殊采购模式

（1）VMI（供应商管理库存）：联想是国内IT企业中最早实施VMI的企业之一（2004年启动）。通过与供应商的可视化库存协同管理，实时监测库存水平，使库存周转天数从原来的14天缩减到5天，库存成本降低60%。

（2）CTO（客户定制，Configure To Order）：针对客户定制需求，建立了灵活的配置系统和快速响应机制。客户可在网上自主选择配置，系统自动生成BOM并分配库存，交付周期从7天缩短至3天。

（3）电子招标：通过电子招标平台开展采购活动，实现供应商在线竞价。2023年通过电子招标节约采购成本约12亿元人民币。

四、采购管理现状评价

（一）采购部门设置

联想集团的采购管理采用"集中决策、分散执行"的组织模式：

1. 全球采购中心（GPO，Global Procurement Organization）

设在美国北卡罗来纳州，负责战略寻源、供应商管理、采购策略制定等职能。下设：
- 生产采购团队（Direct Procurement）：负责原材料和零部件采购，约150人
- 一般采购团队（Indirect Procurement）：负责非生产性物料采购，约80人
- 供应商管理团队（SRM）：负责供应商关系维护和绩效评估，约30人

2. 区域采购团队

在中国（北京、上海、深圳）、美国、巴西、匈牙利、印度等地设立区域采购中心，负责本地采购执行、供应商协调、物流管理等操作性工作。中国区采购团队约200人。

3. 跨职能协同机制

通过"品类委员会"（Commodity Council）机制，每月召开跨部门会议，参与方包括采购、研发、质量、财务等部门，共同决策品类战略。

（二）人员配置与能力要求

根据对23份招聘JD的分析，联想采购人员分为三个层级：

1. 采购专业人员（占比60%）
- 岗位：采购工程师、供应商质量工程师（SQE）、物料计划员
- 要求：本科及以上学历，3-5年相关经验，熟悉ERP系统（SAP）、精通Excel数据分析
- 薪酬：年薪15-25万元（中国区）

2. 战略采购人员（占比25%）
- 岗位：品类经理（Category Manager）、战略采购经理
- 要求：本科及以上学历，5-8年经验，具备商业谈判能力、成本分析能力、项目管理能力
- 薪酬：年薪30-50万元

3. 管理支持人员（占比15%）
- 岗位：采购系统分析师、流程改进专员、合规审计师
- 要求：数据分析能力、熟悉SQL/Python、六西格玛认证优先
- 薪酬：年薪25-40万元

（三）采购管理优势

1. 数字化转型行业领先

联想自主研发了供应链智能控制塔（SCI）系统，打通了供应商的信息系统，将计划、采购、制造、交付和质量控制全流程集成至一个平台进行协同。超过70%的联想供应链员工使用该系统，每天处理1500个数据任务。SCI系统使收入增长了4.8%，准时交付率提升5%，制造和物流成本降低约20%。

2. 绿色供应链体系完善

联想集团制定了体系化的控制措施和程序来管理整个采购流程，将ESG管控措施整合到主采购流程中。自主开发了绿色供应链数据管理平台（GSCDM），从供应链的多个层面收集和分析材料信息和数据。2023/24财年，95%的供应商（按采购金额计）通过RBA审核，80%的生产采购支出来自本地供应商（减少碳排放）。

3. 供应商协同能力强

通过供应商门户系统（Supplier Portal），实现了与供应商的在线协同，包括：在线报价提交、订单状态实时查询、库存信息共享（VMI）、质量数据交换、电子对账等。系统使用率达90%，供应商满意度评分4.2/5.0。

4. 成本控制效果显著

通过实施VMI、电子招标、集中采购、标准化设计（Design for Cost）等措施，有效降低了采购成本和库存成本。2023年采购成本节约率达8.5%，库存周转率提升至12次/年（行业平均8-10次）。

五、采购过程中遇到的挑战和解决方案

（一）主要挑战

1. 价格波动风险大

IT行业原材料价格波动剧烈，半导体、显示屏等核心零部件价格受市场供需关系影响大。以DRAM内存为例，2021-2023年价格波动幅度超过200%。同时，国际汇率波动（美元兑人民币汇率年波动5-8%）、地缘政治（如中美贸易摩擦导致的芯片出口管制）等因素也带来价格不确定性。2023年因芯片短缺导致的紧急采购成本增加约5亿元人民币。

2. 技术迭代速度快

电子产品更新换代频繁，核心零部件（如CPU、内存）的技术迭代周期仅12-18个月，旧型号快速贬值。例如，Intel CPU每年更新换代，上一代产品降价幅度达30-50%。这给采购计划和库存管理带来巨大挑战，采购过早导致库存贬值，采购过晚面临缺货风险。

3. 需求差异化与标准化矛盾

既要满足客户个性化定制需求（CTO订单占比达40%），又要保持规模效应和成本控制。过度标准化导致产品同质化，竞争力下降；过度定制化导致采购批量小、供应商配合度低、成本上升。如何在标准化与差异化之间取得平衡是重要挑战。

4. 供应链复杂性高

联想拥有300多家供应商，分布在全球30多个国家，涉及海运、空运、陆运等多种运输方式和海关流程。任何一个环节出现问题（如2021年苏伊士运河堵塞、2022年上海疫情封控）都会影响全球供应链。供应链可视化程度有限，对二级、三级供应商的管控能力弱。

5. 可持续发展压力

随着环保法规日益严格（如欧盟碳边境调节机制CBAM）和消费者环保意识提升，如何在采购环节落实ESG要求、实现绿色供应链管理成为新挑战。部分供应商环保意识薄弱，绿色改造成本高，推行难度大。

（二）挑战的深层原因分析

1. 价格波动大的根本原因：产业链地位与周期性

半导体行业具有典型的"周期性"特征，受摩尔定律驱动，技术迭代导致旧型号快速贬值。联想作为下游组装厂，处于产业链"微笑曲线"中段，利润空间被上游芯片厂商（如Intel、AMD、NVIDIA）和下游渠道商挤压，缺乏定价权。同时，半导体产能扩张周期长（2-3年），供需错配导致价格剧烈波动。

2. 需求差异化矛盾的本质：规模经济与范围经济的冲突

这是制造业的普遍矛盾。标准化有利于降低采购成本（规模效应）、减少库存品种、提高生产效率；但PC市场已进入成熟期，产品同质化严重，差异化成为竞争关键。联想通过CTO模式实现了两者的平衡，但牺牲了部分规模效应，导致采购成本比纯标准化生产高3-5%。

3. 供应链复杂性的根源：全球化布局与风险分散

为降低劳动力成本和贴近市场，联想在全球布局生产基地（中国、美国、墨西哥、巴西、匈牙利、印度），这必然带来供应链的复杂性。虽然多元化布局分散了单一地区风险，但也增加了协调难度。此外，为降低供应风险而采取的多供应商策略（同一物料2-3家供应商），也增加了管理复杂度。

（三）解决方案与实施效果

1. 建立一体化运作体系

将采购、生产、分销以及物流整合成一个统一的系统，在整个集团层面实行统一策略、统一协调。通过ERP系统（SAP）和供应链管理系统（SCM）实现信息共享和流程协同。实施效果：订单处理时间缩短40%，跨部门沟通效率提升30%。

2. 采用混合生产模式（安全库存+按订单生产）

采取安全库存结合按订单生产的方式，保持1-2天成品安全库存（针对畅销型号），同时根据用户订单快速响应市场需求。这样既保证了交付速度，又避免了过多库存积压。实施效果：库存周转天数从18天降至12天，缺货率从5%降至2%。

3. 实施供应商管理库存（VMI）

与核心供应商（占采购额60%的TOP20供应商）建立VMI合作关系，由供应商管理库存水平，根据实际需求实时补货。联想通过供应商门户系统实时查看供应商库存，系统自动触发补货指令。实施效果：库存周转天数从14天降低到5天，库存成本降低60%，缺货率降低至1%以下。

4. 构建智能预测系统（SCI）

利用AI和大数据技术，建立供应链智能控制塔（SCI）系统，整合销售预测、库存数据、供应商信息、宏观经济指标等，通过机器学习算法提高预测准确性。系统可自动识别风险（如某供应商交货延迟）并给出应对建议（如启用备选供应商）。实施效果：预测准确率从75%提升至85%，决策时间缩短50-60%。

5. 推进绿色采购与ESG管理

（1）制定《联想绿色采购标准》，要求供应商签署《供应商行为准则》，承诺遵守RBA行为准则。

（2）建立绿色供应链数据管理平台（GSCDM），追踪产品全生命周期的环境影响，包括碳足迹、水足迹、有害物质使用等。

（3）优先选择本地供应商，减少物流碳排放。目前中国区80%的生产采购支出来自本地供应商，平均运输距离缩短30%。

（4）实施供应商ESG评级，将评级结果与订单分配挂钩。A级供应商可获得优先付款、更多订单份额等激励。

实施效果：2023/24财年供应链碳排放强度降低15%，获得CDP"供应链脱碳先锋奖"（中国唯一获奖企业）。

6. 加强供应商协同与早期参与

（1）建立供应商门户系统（Supplier Portal），实现在线协同和信息共享，包括设计图纸共享（PLM系统对接）、质量数据实时交换、电子对账等。

（2）实施"早期供应商参与"（ESI, Early Supplier Involvement），在产品研发阶段（概念设计阶段）就邀请关键供应商参与，共同进行可制造性设计（DFM）和可采购性设计（DFP），从源头降低成本。

（3）建立持续改进机制，每季度与战略供应商召开QBR（季度业务回顾）会议，通过绩效评估（质量、成本、交付、服务、技术五维度评分）和反馈促进供应商能力提升。

实施效果：供应商准时交付率从90%提升至95%，质量问题投诉减少40%，联合创新项目增加30%。

7. 建立供应链风险管理体系

（1）建立关键物料（芯片、面板）价格波动监测体系，设置价格预警线（如较上月波动超过15%触发预警），触发套期保值或战略采购。

（2）对TOP50关键物料实施"双源供应"策略，主供应商占比不超过70%，备选供应商占比30%，并定期进行切换演练。

（3）建立战略库存（Strategic Stock），对供应风险高的芯片建立6个月安全库存，应对地缘政治风险。

（4）建立供应链韧性评估体系，定期评估供应商的财务稳定性、产能弹性、替代能力等。

实施效果：2022年芯片短缺期间，通过提前储备和备选供应商切换，保障了95%的订单交付，而行业平均水平仅为80%。

六、总结与建议

（一）总结

通过对联想集团采购流程的深入调研分析，可以得出以下结论：

1. 坚持一体化运作理念，将采购纳入整体供应链体系进行统筹管理，实现了从战略到执行的统一协调。

2. 积极推进数字化转型，自主研发的SCI系统处于行业领先水平，利用AI、大数据等技术提升了供应链智能化水平。

3. 重视供应商关系管理，通过VMI、ESI、QBR等机制与核心供应商建立了长期战略合作关系。

4. 践行可持续发展理念，建立了完善的绿色供应链体系，将ESG要求融入采购全流程，获得国际权威认可。

5. 成本控制能力强，通过电子招标、VMI、标准化设计等措施，实现了行业领先的采购成本节约率（8.5%）。

（二）改进建议（含实施方案）

1. 建立供应链风险预警与应急响应机制（2026年Q2前完成）

责任部门：全球供应链风险管理部（新建）

具体措施：
- 建立7×24小时供应链监控中心，实时监测全球供应商产能、物流、政策变化
- 开发供应链风险预警AI模型，提前30天预测供应中断风险（准确率目标>80%）
- 制定分级应急响应预案（黄色/橙色/红色三级），明确启动条件和响应流程
- 每半年进行一次供应链中断模拟演练（如"某关键芯片厂火灾"场景）

考核指标：供应中断响应时间<24小时，中断恢复时间<72小时，年度供应中断次数≤2次

预期收益：供应链韧性提升，潜在损失减少50%

2. 推进采购数字化转型二期工程（2026-2027年）

责任部门：IT部+采购部（联合项目组）

具体措施：
- 引入区块链技术，建立"可信供应链"平台，实现供应商资质、交易记录、质检报告不可篡改，提高透明度
- 开发AI谈判助手（Negotiation Bot），自动分析供应商报价历史、市场行情，生成谈判策略建议，预计可提升谈判效率30%
- 建立供应商协同云平台，实现设计图纸（CAD文件）实时共享、在线协同设计（Co-design）
- 推广RPA（机器人流程自动化），实现采购订单自动生成、对账自动化，减少人工操作80%

投资预算：5000万元人民币

预期收益：采购周期缩短20%，人工成本降低15%，采购错误率降低至0.1%以下

3. 深化上游供应商合作，建立战略联盟（2026-2028年）

责任部门：战略采购部

具体措施：
- 与TOP5战略供应商（占采购额40%）建立"联合创新中心"，共同投资研发新材料、新工艺，共享知识产权
- 实施"供应商能力提升计划"，每年投入5000万元帮助关键供应商进行数字化改造和绿色升级
- 建立长期协议（LTA）+价格共享机制，与供应商共担原材料价格波动风险，增强合作黏性
- 延伸到二级、三级供应商（原材料供应商），建立多级供应商管理平台，实现全链条可视化

考核指标：战略供应商合作年限>5年占比提升至60%，联合创新项目≥10个/年

预期收益：供应链稳定性提升，独家供应风险降低，创新速度加快

4. 加强采购人员能力建设（持续进行）

责任部门：人力资源部+采购培训学院

具体措施：
- 建立"采购人才梯队"，设立采购专员-高级采购-采购专家-首席采购官（CPO）四级职业发展通道
- 实施"数字化采购能力认证"，要求所有采购人员掌握数据分析（SQL/Python）、BI工具（Tableau）、RPA操作
- 每年选派20名优秀采购人员赴MIT、斯坦福等名校进修供应链管理课程
- 建立"采购案例库"，沉淀最佳实践，供全员学习

考核指标：采购人员数字化能力认证通过率100%，人均培训时长>40小时/年

预期收益：采购团队专业能力提升，战略采购人员占比从25%提升至40%

5. 探索采购新模式：从"成本中心"到"价值中心"转型（2026-2029年）

责任部门：采购部+财务部（联合）

具体措施：
- 建立"采购价值核算体系"，不仅核算采购成本节约，还要核算采购对营收的贡献（如通过快速交付赢得的订单）
- 推行"品类管理2.0"，从"买得便宜"转向"买得对"，参与产品前期设计，通过VA/VE（价值分析/价值工程）降低产品总成本
- 建立"供应商创新基金"，每年投入1亿元，奖励提出创新方案的供应商，共享降本收益
- 拓展采购职能，从"物料采购"延伸到"服务采购""能力采购"（如购买AI算法、云服务）

考核指标：采购贡献的营收占比>5%，VA/VE降本金额占总采购额3%

预期收益：采购部门从成本中心转型为价值中心，在企业战略中的地位提升
